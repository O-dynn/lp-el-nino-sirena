# 🌊 El Niño Sirena — Case Acadêmico

Case visual desenvolvido para apresentar o projeto **El Niño Sirena**, explorando duas interpretações gráficas para o mesmo título: uma versão voltada para **audiobook** e outra para **livro infantil**.

O objetivo do projeto é demonstrar como um mesmo universo narrativo pode assumir atmosferas visuais diferentes por meio de cor, composição, tipografia e linguagem gráfica.

🔗 Visualização online no portfólio:
https://o-dynn.github.io/lp-el-nino-sirena/

---

## 🧠 Sobre o Projeto

**El Niño Sirena** foi concebido como um estudo de direção visual aplicado à capa editorial.

A proposta parte de duas leituras:

- **Audiobook:** abordagem mais profunda, silenciosa e sensorial  
- **Livro infantil:** composição mais leve, colorida e lúdica  

A estrutura do case foi desenvolvida para funcionar como peça de portfólio autoral, destacando não apenas o resultado final, mas também o raciocínio visual por trás das decisões de design.

---

## 🎨 Conceito Visual

O projeto trabalha o contraste entre duas atmosferas dentro de um mesmo universo submarino:

- profundidade e contemplação  
- cor e leveza  
- mistério e aproximação  

A identidade visual do site acompanha esse conceito com:

- fundo escuro em tons azulados  
- contraste de textos claros  
- painéis translúcidos  
- composição inspirada em profundidade oceânica  

---

## 🚀 Tecnologias Utilizadas

- **HTML5** — estrutura semântica e organização do case  
- **CSS3** — composição visual, responsividade e direção estética  
- **JavaScript (ES6)** — interações, modal e animações suaves  
- **Intersection Observer** — revelação progressiva de conteúdo
